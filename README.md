# Blog using ReactJS and Redux

Using Router5 and bootstrapped with Webpack2.

## How to

**Install dependencies**
```bash
yarn
```
**Run development environment**
```bash
yarn start
```
**Build the development bundle**
```bash
yarn run build-dev
```
**Build the production bundle**
```bash
yarn run build-prod
```
**Run linter**
```bash
yarn run lint
```
**Run tests**
```bash
yarn run test
```
